# 🎵 Magic Shop - my_favourite_music_Page

A beginner-friendly BTS fan website created using **HTML, CSS, Bootstrap, and CCBP UI Kit**.

This project showcases some of my favorite BTS songs with song information, images, videos, and interactive navigation between pages.

---

## ✨ Features

- 🏠 Landing page with a beautiful background image
- 🎶 List of favorite BTS songs
- 📸 Song cover images and album artwork
- 🎥 Embedded YouTube music videos
- 🖼️ Image carousel for selected songs
- 📱 Responsive design for mobile and desktop screens
- 🔄 Smooth navigation between sections

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- CCBP UI Kit
- YouTube Embedded Videos
